Documento de Ayuda del Cliente

1. Funcionalidades del cliente

1.1 Registro y Autentificación

1.1.1 Crear una cuenta

Si el usuario aún no tiene una cuenta, el usuario puede registrarse presionando en el botón registrar (Ver figura 1).

Figura 1: Crear una cuenta

A continuación, se mostrará un formulario donde el cliente debe ingresar sus datos y dar clic en el botón Sing Up. Cabe recalcar que un cliente no se puede registrar más de una vez, si esto llegara a suceder saltara un mensaje de error.

Figura 2: Registro del cliente

1.1.2 Iniciar sesión

Una vez que el cliente ya se ha registrado puede iniciar sesión con su correo electrónico y contraseña (Ver Figura 3).

Figura 3: Iniciar sesión cliente

1.2 Exploración del catálogo del producto

Al presionar el botón Catalogo el cliente puede ver todos los productos que están categorizados por estilo.

Figura 4: Catálogo

1.3 Compra

Al presionar el botón comprar se abre una página donde el cliente puede realizar la compra de uno o varios productos. El cliente puede elegir el mueble dependiendo del estilo como se puede observar en la Figura 5.

Figura 5: Compra de muebles

Una vez que el cliente haya elegido uno o varios muebles para la compra, se debe presionar el botón vender donde reflejará el monto a pagar, y presionar el botón pagar (Ver Figura 6).

Figura 6: Comprar producto
