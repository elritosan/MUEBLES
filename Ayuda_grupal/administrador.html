Documento de Ayuda Administrador

1. Pantalla de Inicio y Menú del Administrador

1.1 Descripción de la pantalla de inicio

La página de inicio del administrador es acogedora y sofisticada, destacando un eslogan que resalta la elegancia y el confort que sus muebles aportan al hogar. El diseño visual incluye una imagen atractiva de una sala bien decorada, transmitiendo el estilo y calidez de la aplicación. En el centro de la página se encuentra el logotipo de LuxuryMuebles, consolidando la identidad visual de la empresa.

Figura 1: Página de inicio del administrador

En la parte superior de la página, el menú de navegación permite al administrador acceder rápidamente a diferentes funciones clave como ingresar, editar, eliminar y buscar muebles. Además, pueden generar reportes detallados.

1.2 Elementos del Menú del administrador

  • Ingresar nuevo mueble
  • Eliminar mueble
  • Reportes sobre muebles y clientes
  • Editar datos del mueble
  • Buscar muebles
  • Cierra la página del administrador

Figura 2: Menú del administrador

2. Pantalla de Inicio del Cliente

2.1 Descripción de la pantalla de Inicio

La página de Inicio del cliente es igual a la del administrador con la excepción del menú para más información léase la descripción de la pantalla de inicio del administrador.

Figura 3: Página de inicio del cliente

2.2 Elementos del Menú del cliente

  • Comprar
  • Perfil del cliente
  • Catálogo de los diferentes muebles
  • Botón para Iniciar sesión
  • Cierra la página del cliente

Figura 4: Menú del cliente

3. Funcionalidades del administrador

3.1 Iniciar sesión

3.1.1 Acceso a la cuenta del administrador

En esta pantalla de iniciar sesión el administrador debe ingresar sus credenciales de correo y contraseña y presionar sobre el botón Iniciar sesión.

Figura 5: Inicio de sesión del administrador

3.2 Gestión de muebles

3.2.1 Agregar Mueble

Al presionar en el botón Ingresar del menú se abre un formulario en la cual podemos ingresar datos para un nuevo mueble.

Figura 6: Ingresar

Figura 7: Ingreso de datos de un mueble

Una vez llenado todos los datos se debe presionar el botón guardar, o si cambia de parecer y ya no quiere ingresar ningún mueble, presionar el botón cancelar.

Figura 8: Guardar Mueble

Cabe indicar que no se puede ingresar el mismo código de un mueble que ya ha sido ingresado, en caso de hacerlo saltará un mensaje de error.

Figura 9: Error de ingreso de código

3.2.2 Buscar Mueble

Al presionar en el botón Buscar del menú se abre un formulario en la cual podemos ingresar el código para buscar el mueble.

Figura 10: Buscar mueble

Una vez ingresado el código se debe presionar el botón BUSCAR y se mostrará toda la información del mueble.

Figura 11: Mostrar datos del buscar

3.2.3 Editar Mueble

Al presionar en el botón Editar del menú se abre un formulario en la cual podemos ingresar el código o el nombre del mueble que se desea editar.

Figura 12: Buscar para editar

Una vez ingresado el código del mueble o el nombre del mueble se presiona el botón buscar donde se mostrarán todos los datos, y se puede editar. Cabe recordar que una vez que se haya elegido el campo por el cual se quiere buscar el mueble el otro campo desaparece.

Figura 13: Elegir campo

Figura 14: Editar datos

Ya editado los datos se presiona el botón Editar para que se guarden los datos, o si cambia de parecer presiona el botón cancelar para que no se edite ningún dato.

Figura 15: Botón editar

3.2.4 Eliminar Mueble

Al presionar en el botón Eliminar del menú se abre un formulario en la cual podemos ingresar el código del mueble que se desea eliminar.

Figura 16: Buscar mueble para eliminar

Una vez ingresado el código se debe presionar el botón buscar posterior se mostrarán todos los datos del mueble si este existe, y presionar en el botón eliminar.

Figura 17: Eliminar mueble
